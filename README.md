📊 Retail Sales AnalysisAn end-to-end SQL data analysis project exploring retail sales trends, customer behavior, and product performance. Features optimized SQL queries, data cleaning, and actionable insights to drive business strategy.

🛠️ Tech StackDatabase: SQL (PostgreSQL / MySQL / SQL Server)

Concepts: CTEs, Window Functions, Joins, Aggregations

🚀 Key Features & QueriesData Cleaning: Handling missing values and formatting date fields.Sales Trends: Identifying peak shopping hours, monthly revenue, and yearly growth.Customer Behavior: Segmenting customers based on purchase frequency and total spend.Product Insights: Analyzing top-selling categories and lowest-performing items.
